## 故障排查

### 问题 1：草稿文件未创建

**检查：**
```bash
# 查看日志
tail -f "C:\Users\Administrator\AppData\Local\Storyboard\logs\app-*.log"
```

**可能原因：**
- 项目路径查询失败
- 权限问题
- 磁盘空间不足

### 问题 2：时间轴不显示

**检查日志：**
```
[INFO] 草稿未加载，尝试自动加载...
[WARN] 无法获取项目信息，跳过草稿加载
```

**解决：**
- 确保 `GetCurrentProjectPathQuery` 消息处理器已注册
- 确保 `GetProjectInfoQuery` 消息处理器已注册

### 问题 3：JSON 文件为空

**检查：**
- 是否有镜头生成了视频？
- 视频文件路径是否存在？
- 查看日志中的同步信息
