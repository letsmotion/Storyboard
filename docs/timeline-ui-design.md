## 详细说明

### 1. 顶部工具栏
- 左侧：项目操作（导入、导出、撤销、重做）
- 中间：播放控制（播放、暂停、停止）
- 右侧：当前时间码显示

### 2. 视频预览播放器
- 占据上半部分，显示当前播放头位置的帧
- 支持缩放、全屏预览
- 显示安全框、网格线等辅助线（可选）

### 3. 播放控制栏
- 上一帧/下一帧按钮（精确控制）
- 播放/暂停按钮
- 进度条（可拖动，与时间轴同步）

### 4. 时间轴工具栏
- 添加轨道按钮
- 缩放滑块（调整时间轴显示密度）
- 适应窗口按钮

### 5. 时间轴主体

#### 左侧轨道控制列
```
┌──────┐
│ 🎬   │ ← 轨道类型图标
│视频1 │ ← 轨道名称（可编辑）
│ 🔒🙈 │ ← 锁定/隐藏按钮
└──────┘
```

#### 右侧时间轴区域
- **时间刻度尺**：显示时间标记（0s, 1s, 2s...）
- **播放头**：红色竖线，贯穿所有轨道，可拖动
- **视频片段**：
  ```
  ┌──────────┐
  │ Shot #1  │ ← 片段编号
  │ [缩略图] │ ← 首帧缩略图
  │  3.5s    │ ← 时长显示
  └──────────┘
  ```
- **占位符**（未生成视频）：
  ```
  ┌──────────┐
  │ Shot #3  │
  │ [📹待生成]│ ← 占位图标
  │  4.0s    │
  └──────────┘
  ```

## 交互功能

### 片段操作
- **拖动**：左右移动片段位置
- **调整时长**：拖动片段边缘
- **分割**：点击片段，选择分割点
- **删除**：选中后按 Delete 或右键菜单
- **复制/粘贴**：Ctrl+C / Ctrl+V

### 播放头操作
- **拖动**：直接拖动播放头到任意位置
- **点击时间轴**：播放头跳转到点击位置
- **键盘控制**：
  - 空格：播放/暂停
  - ← →：上一帧/下一帧
  - J K L：倒放/暂停/正放

### 缩放操作
- **滑块**：调整时间轴缩放级别
- **鼠标滚轮**：Ctrl + 滚轮缩放
- **适应窗口**：自动调整缩放以显示全部内容

### 轨道操作
- **添加轨道**：点击"添加轨道"按钮
- **删除轨道**：右键轨道名称 → 删除
- **锁定轨道**：防止误操作
- **隐藏轨道**：临时隐藏不需要的轨道

## 视觉设计要点

### 配色方案（深色主题）
- 背景：`#0a0a0a`（深黑）
- 轨道背景：`#1a1a1a`（浅黑）
- 片段背景：`#27272a`（灰色）
- 选中片段：`#3b82f6`（蓝色边框）
- 播放头：`#ef4444`（红色）
- 时间刻度：`#71717a`（灰色文字）

### 片段样式
```css
片段容器 {
  背景: #27272a
  边框: 2px solid #3f3f46
  圆角: 4px
  内边距: 8px
}

片段:hover {
  边框: 2px solid #6366f1
  阴影: 0 0 8px rgba(99, 102, 241, 0.3)
}

片段:selected {
  边框: 2px solid #3b82f6
  阴影: 0 0 12px rgba(59, 130, 246, 0.5)
}
```

### 缩略图显示
- 显示首帧和尾帧（左右各一个）
- 尺寸：根据片段宽度自适应
- 占位符：显示图标 + 文字提示

## 响应式布局

### 窗口宽度 > 1200px
- 预览播放器：占 60% 高度
- 时间轴：占 40% 高度

### 窗口宽度 800-1200px
- 预览播放器：占 50% 高度
- 时间轴：占 50% 高度

### 窗口宽度 < 800px
- 预览播放器可折叠
- 时间轴占据主要空间

### 数据模型
```csharp
public class TimelineTrack
{
    public Guid Id { get; set; }
    public string Name { get; set; }
    public TrackType Type { get; set; } // Video, Audio, Subtitle
    public bool IsLocked { get; set; }
    public bool IsVisible { get; set; }
    public ObservableCollection<TimelineClip> Clips { get; set; }
}

public class TimelineClip
{
    public Guid Id { get; set; }
    public Guid ShotId { get; set; }
    public double StartTime { get; set; }
    public double Duration { get; set; }
    public ClipStatus Status { get; set; } // Generated, Generating, Placeholder
    public string ThumbnailPath { get; set; }
    public string VideoPath { get; set; }
}

public enum ClipStatus
{
    Placeholder,    // 未生成
    Generating,     // 生成中
    Generated,      // 已生成
    Error          // 生成失败
}
```
