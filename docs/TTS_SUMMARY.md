# TTS 功能现状总结（2026-03-05）

## 总体结论

TTS 功能已完成从服务层到 UI 的主链路打通，当前用户可在镜头编辑器中单镜头生成配音，也可在镜头列表中批量生成配音。  
下一阶段重点不再是“有没有功能”，而是“体验完善 + 数据持久化 + 音频编排能力”。

## 实现状态总览

| 模块 | 状态 | 说明 |
| --- | --- | --- |
| TTS Provider 抽象 | ✅ 已完成 | `ITtsProvider`、请求/结果模型已落地 |
| NewAPI Provider | ✅ 已完成 | 已接入 `/v1/audio/speech`，支持音色/语速/格式 |
| TTS 服务层 | ✅ 已完成 | 支持单镜头、批量生成、提供商选择 |
| 单镜头 UI | ✅ 已完成 | `ShotEditorView.axaml` 的“配音生成”标签可用 |
| 批量配音 UI | ✅ 已完成 | 已有批量对话框与进度、失败统计 |
| 导出链路（CapCut 草稿） | ⚠️ 部分完成 | 可读取 `GeneratedAudioPath` 注入音轨 |
| 时间轴音频编辑 | ❌ 未完成 | 暂无音频轨道与波形编辑能力 |
| 内置播放器 | ⚠️ 部分完成 | 镜头编辑器已内置播放器；其余入口待统一 |
| 工程数据持久化 | ✅ 已完成 | `ShotState` 已落地并完成保存/读取映射 |

## 已完成能力

1. 服务能力
   - `GenerateAsync`：基础文本转语音。
   - `GenerateForShotAsync`：按镜头输出到固定目录。
   - `GenerateBatchAsync`：按镜头顺序生成，单条失败不阻塞后续。

2. 单镜头 UI 能力
   - 输入配音文本、选择音色、语速控制。
   - 生成/上传/内置播放/删除操作完整可用。
   - 生成状态与错误信息可见。
   - 播放支持播放/暂停/停止、进度与时间显示。

3. 批量 UI 能力
   - 入口：镜头列表工具栏“批量配音”按钮。
   - 选项：统一音色、语速、跳过已有音频、优先使用已有配音文本。
   - 结果：成功/失败/跳过统计，失败明细展示。

## 当前限制与风险

1. 播放器覆盖范围不完整  
   当前仅镜头编辑器内实现内置播放，其他播放入口仍需统一。

2. 音频编辑能力仍不足  
   暂无波形可视化、裁剪、淡入淡出、多轨混音等能力。

## 下一阶段建议（优先级）

### P0
- 统一所有播放入口到同一套内置播放器体验。

### P1
- 时间轴音频轨道与镜头时长同步策略。

### P2
- 音频混音与导出参数（音量、淡入淡出、多轨）。
- 扩展更多 TTS Provider（Qwen/Volcengine）。
